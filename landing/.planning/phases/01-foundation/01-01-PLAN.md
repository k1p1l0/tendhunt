  <action>
Create the Astro 5.x project from scratch inside the `landing/` directory (the directory already exists but is empty except for `.planning/`).

**Step 1: Initialize project**
Run `npm create astro@latest -- --template minimal --no-install --no-git .` from within `landing/` to scaffold in the current directory. Then run `npx astro add tailwind` to install `@tailwindcss/vite` (answer yes to all prompts). If `npx astro add tailwind` doesn't work cleanly, manually install: `npm install @tailwindcss/vite` and configure the Vite plugin in astro.config.mjs.

**Step 2: Install dependencies**
```bash
npm install @fontsource-variable/space-grotesk @fontsource-variable/inter astro-navbar astro-seo sharp @astrojs/sitemap
npm install -D wrangler prettier prettier-plugin-astro
```

**Step 3: Configure astro.config.mjs**
```javascript
import { defineConfig } from "astro/config";
import tailwindcss from "@tailwindcss/vite";
import sitemap from "@astrojs/sitemap";

export default defineConfig({
  site: "https://tendhunt.com",
  output: "static",
  vite: {
    plugins: [tailwindcss()],
  },
  integrations: [sitemap()],
});
```

CRITICAL anti-patterns to avoid:
- Do NOT create `tailwind.config.js` -- Tailwind v4 uses `@theme` in CSS, not a config file
- Do NOT use `@astrojs/tailwind` -- that's the deprecated v3 integration
- Do NOT use `@astrojs/cloudflare` adapter -- not needed for static sites
- Do NOT use Google Fonts @import -- use Fontsource self-hosted packages

**Step 4: Create src/styles/global.css with design tokens**
Use the exact `@theme` block from the RESEARCH.md code example. This defines:
- Font families: `--font-heading: "Space Grotesk Variable"`, `--font-body: "Inter Variable"`
- Colors (dark theme): `--color-bg-primary: #0A0A0A`, `--color-bg-secondary: #111111`, `--color-bg-tertiary: #1A1A1A`, `--color-bg-card: #161616`, `--color-bg-light: #F5F5F5`, `--color-bg-light-secondary: #EBEBEB`
- Text colors: `--color-text-primary: #FFFFFF`, `--color-text-secondary: #A1A1A1`, `--color-text-body: #D4D4D4`, `--color-text-muted: #737373`, `--color-text-dark: #171717`
- Accent: `--color-accent: #E5FF00`, `--color-accent-hover: #D4EE00`, `--color-accent-muted: #E5FF0033`
- Borders: `--color-border: #2A2A2A`, `--color-border-light: #E5E5E5`
- Container: `--container-content: 80rem` (1280px per user decision)
- Section spacing: `--spacing-section: 6rem`, `--spacing-section-sm: 4rem`, `--spacing-section-lg: 8rem`

Include base HTML styles: `scroll-behavior: smooth`, `-webkit-font-smoothing: antialiased`.
